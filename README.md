# 🚀 Nova AI

> **Your AI-Powered Productivity Companion**

Nova AI is an AI-powered productivity platform designed to help **students, professionals, and entrepreneurs** plan, prioritize, and complete tasks before deadlines are missed.

Unlike traditional to-do applications, Nova AI actively analyzes workload, predicts deadline risks, generates intelligent schedules, and provides AI-powered recommendations to help users make better decisions and stay productive.

---

## 🌟 Problem Statement

Students, professionals, and entrepreneurs frequently miss assignments, meetings, bill payments, interviews, and other important commitments. Existing productivity tools rely heavily on passive reminders that are easy to ignore.

**Nova AI** solves this problem by acting as an AI productivity companion that proactively helps users complete tasks before deadlines are missed.

# 🚀 Installation


---

# 🔑 Environment Variables

Create a `.env` file and configure:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY

VITE_FIREBASE_API_KEY=YOUR_FIREBASE_API_KEY

VITE_FIREBASE_AUTH_DOMAIN=YOUR_AUTH_DOMAIN

VITE_FIREBASE_PROJECT_ID=YOUR_PROJECT_ID

VITE_FIREBASE_STORAGE_BUCKET=YOUR_STORAGE_BUCKET

VITE_FIREBASE_MESSAGING_SENDER_ID=YOUR_SENDER_ID

VITE_FIREBASE_APP_ID=YOUR_APP_ID
```
